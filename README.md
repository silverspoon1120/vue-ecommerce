## Key Features ‚ú®

* **50+** reusable components
* **Small** `~7KB` (minified and gzipped)
* Fully **tree-shakeable**
* **No dependencies**
* **No SDK**
* **Don't track** the user
* Built with **a11y** in mind
* **One API** to rule them all
* **i18n** support
* **TypeScript** support
